Nuevas maneras de infectar tu equipo y como defenderte de ellas.



Expertos en ciberseguridad han detectado nuevas técnicas de engaño utilizadas por ciberdelincuentes para evitar ser detectados. Estas técnicas incluyen ocultar malware en:
Archivos PDF muy realistas, que simulan documentos legítimos y pueden contener scripts maliciosos.
Imágenes pixeladas, que disfrazan el código malicioso como si fueran archivos de diseño o proyectos.
Archivos comprimidos en formato IMG, que se usan para instalar programas maliciosos como Lumma Stealer.
También se están usando herramientas legítimas del sistema, como PowerShell, para ejecutar comandos maliciosos sin levantar sospechas. Los atacantes prefieren archivos comprimidos como .rar para distribuir el malware, ya que son menos detectados por los sistemas de seguridad.
